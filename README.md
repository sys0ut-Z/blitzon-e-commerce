# Blitzon

A MERN stack gaming gear e-commerce application with authentication, online payments, and admin dashboard.

## 🚀 Features
- JWT Authentication
- Add to Cart
- Cash on Delivery
- Stripe Payment
- Admin Dashboard(on local machine, not deployed)
- Order Management

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## 🔗 Live Demo
Deployed on [Vercel](https://blitzon-e-commerce.vercel.app/)
