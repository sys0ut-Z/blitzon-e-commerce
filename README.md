# Blitzon

A MERN stack gaming gear e-commerce application with authentication, online payments, and admin dashboard.

## 🚀 Features
- JWT Authentication
- Add to Cart
- Cash on Deivery
- Stripe Payment
- Admin Dashboard(local, not deployed)
- Order Management

## 🛠️ Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## 🔗 Live Demo
Deployed on [Vercel](https://blitzon-e-commerce.vercel.app/)
